# 电商用户价值分层与精准营销策略报告

## 1. 数据与建模口径

- 样本量：1000 名用户。
- 传统 RFM：R=最近登录越近越高，F=购买次数越多越高，M=总消费越高越高；综合分 = 0.35R + 0.30F + 0.35M。
- 新增意向度：I-score = 停留时长分位分 * 0.5 + 浏览页数分位分 * 0.5，用于衡量“想不想买”。
- 新增转化摩擦系数：浏览页数 / (购买次数 + 1)，越高代表看得多、买得少，决策更纠结。
- 新增活跃连接度：订阅且 7 天内登录为高连接；未订阅但 7 天内登录为中连接；订阅但 30 天未登录或长期不活跃为低连接；中间状态单独标记为需激活或弱连接。
- 购买力背景不计入综合价值分，只作为修正标签：收入分层 + 消费/收入比例。这样能区分“月薪五万花 1000”和“月薪五千花 1000”的不同购买压力。

增强价值分 = 0.55 * RFM + 0.25 * I-score + 0.20 * (100 - 摩擦分) + 活跃连接修正 + 高/低摩擦修正。

## 3. 重点人群解释与策略

- 纠结土豪：高收入、高意向、低消费且摩擦高。重点不是大水漫灌优惠，而是商品匹配、价格锚点、套装推荐和有限时效优惠，降低“看了很多但没找到合适商品”的决策成本。
- 隐形活跃者：高意向、低收入、低消费且仍保持活跃。更适合会员成长、内容种草、分享返利、裂变奖励，不宜直接推高价商品。
- 高潜流失客：高收入、高消费但近期活跃下降。需要优先触达，询问流失原因，提供专属客服、个性化挽留券或新品优先体验。
- 高潜观望者：本次新增识别，共 5 人。典型表现是停留久、浏览多、近期活跃，但购买少或消费低。策略应以首单/低门槛券、浏览品类的精准推荐、购物车/足迹召回为主。
- 核心用户：增强价值高且近期仍有活跃。适合权益升级、复购券、会员专属推荐，但优惠力度不宜过高，避免补贴本来就会购买的人。

## 4. 分层用户画像与增购策略

### 一般用户

- 用户画像：各项指标居中或特征不明显，是规模最大的人群，适合自动化运营和二次细分。
- 消费阻力：需求、意向、消费能力和活跃度都不突出，单独大额补贴效率低。
- 让他们多花钱的策略：用算法推荐和生命周期任务培养偏好，观察其向潜力、观望或流失方向迁移。
- 推荐权益/优惠：普通品类券、签到积分、浏览后推荐、常规促销。
- 推荐商品或内容：平台热销、个性化猜你喜欢、低风险试购商品。
- 运营 KPI：活跃率、偏好识别率、加购率、向潜力用户迁移率。

### 潜力用户

- 用户画像：综合价值较高，购买和活跃表现尚可，但还没完全稳定为核心用户。
- 消费阻力：对平台信任或购买习惯仍在形成，可能需要更明确的利益点触发下一单。
- 让他们多花钱的策略：围绕最近偏好类目做精准推荐，用阶梯满减和连续购买奖励把偶发消费变成习惯消费。
- 推荐权益/优惠：首选限时满减、第二件优惠、连购奖励、品类券，不建议直接发大额无门槛券。
- 推荐商品或内容：推荐浏览/购买类目的热销款、评价高商品、价格带略高的升级款。
- 运营 KPI：30天复购率、品类渗透率、券后毛利、从潜力转核心的人数。

### 低价值用户

- 用户画像：消费低、意向低或近期不活跃，短期直接拉动消费的性价比较低。
- 消费阻力：需求弱、平台关系弱、价格敏感或并非目标客群。
- 让他们多花钱的策略：控制预算，主要用自动化低成本触达；只有出现高意向行为后再升级运营。
- 推荐权益/优惠：签到、积分、小额低门槛券、清仓品类券。
- 推荐商品或内容：低价爆品、清仓商品、兴趣测试内容。
- 运营 KPI：低成本唤醒率、触达成本、是否向一般/潜力用户迁移。

### 核心用户

- 用户画像：近期活跃、购买基础好、增强价值分最高，是平台当前最稳定的利润来源。
- 消费阻力：不是不买，而是可能被过度补贴；如果只给通用券，会浪费预算并训练其等券消费。
- 让他们多花钱的策略：用会员等级、专属新品、加价购、组合套装和复购周期提醒提升客单价与购买频次。
- 推荐权益/优惠：低折扣高感知权益，如会员日、积分加倍、满额赠、专属客服、预售优先权。
- 推荐商品或内容：推荐高复购、高毛利、升级款、同类目套装和跨品类搭配。
- 运营 KPI：复购率、客单价、毛利率、会员留存、权益使用率。

### 高潜流失客

- 用户画像：收入高、历史消费高，但近期活跃下降，是最需要优先挽回的高价值人群。
- 消费阻力：可能出现价格敏感、体验不满、竞品迁移、需求周期结束或推荐不准。
- 让他们多花钱的策略：先做召回和诊断，再做个性化挽留；对高客单用户给专属服务比单纯降价更有效。
- 推荐权益/优惠：专属召回券、老客回归礼包、客服一对一、包邮/退换无忧、限时高价值权益。
- 推荐商品或内容：围绕历史高消费类目推新品、升级款、补充耗材、同品牌系列。
- 运营 KPI：召回率、回归首单金额、回归后60天留存、流失原因反馈率。

### 沉睡价值用户

- 用户画像：历史价值不低但近期沉睡，仍有唤醒可能，但优先级低于高潜流失客。
- 消费阻力：需求弱化、平台记忆下降、触达疲劳或替代平台迁移。
- 让他们多花钱的策略：用轻触达测试兴趣，再根据点击/回访行为分层加码，避免一开始就高补贴。
- 推荐权益/优惠：回访提醒、兴趣测试券、老客专属小礼包、曾购类目补货提醒。
- 推荐商品或内容：历史购买类目的新品、消耗品补货、季节性商品。
- 运营 KPI：唤醒点击率、回访率、唤醒后购买率、触达退订率。

### 隐形活跃者

- 用户画像：意向高、活跃或半活跃，但收入和消费较低，可能更像平台内容粉丝或价格敏感型用户。
- 消费阻力：购买力有限，对高价商品承接弱；强推高价会降低体验。
- 让他们多花钱的策略：让他们先带来低成本价值：分享裂变、评价晒单、内容互动；消费侧用低价爆品和拼团提升频次。
- 推荐权益/优惠：分享返利、邀请奖励、积分兑换、拼团券、低门槛小额券。
- 推荐商品或内容：推荐平价爆品、试用装、小规格、组合优惠和社交传播素材。
- 运营 KPI：分享人数、邀请新客、低价品复购、内容互动率、裂变 ROI。

### 纠结土豪

- 用户画像：收入高、意向强、浏览多但消费低，典型是有钱也想买，却没找到足够合适或足够确定的商品。
- 消费阻力：选择成本高、商品匹配不足、价格锚点不清、缺少确定性评价或服务保障。
- 让他们多花钱的策略：不要只砸大券，先缩短决策路径：个性化榜单、同价位对比、专家推荐、套装方案，再用小额限时券临门一脚。
- 推荐权益/优惠：高客单品类券、限时小额券、专属导购、试用/无忧退、品质保障。
- 推荐商品或内容：推荐高收入匹配的品质款、品牌款、套装款，以及“帮我选”式内容。
- 运营 KPI：浏览到加购转化率、加购到购买转化率、决策时长、客单价。

### 高潜观望者

- 用户画像：天天逛、停留久、看很多页面，但购买少或不花钱，是新增模型专门识别的高意向未转化人群。
- 消费阻力：可能缺少首单信任、价格门槛、商品选择过载，或者只是把平台当信息查询工具。
- 让他们多花钱的策略：用低门槛首单、足迹召回、购物车提醒和爆品推荐促成首次/下一次购买；目标是先让其完成交易闭环。
- 推荐权益/优惠：首单券、低门槛品类券、限时免邮、浏览商品降价提醒。
- 推荐商品或内容：推荐其高频浏览类目中的爆品、低决策成本商品、评价高且价格适中的商品。
- 运营 KPI：首单转化率、浏览后购买率、券核销率、从观望转潜力的人数。


## 5. 购买力背景修正

购买力不进入主分，避免把“有钱但没需求”误判为高价值；但在运营策略中必须作为标签参与优惠力度和商品价格带选择。

示例：
- #10: 收入 121604, 消费 4999, 消费/收入 4.11%, 标签 高收入/高负担消费
- #863: 收入 101554, 消费 4998, 消费/收入 4.92%, 标签 中高收入/高负担消费
- #198: 收入 114788, 消费 4993, 消费/收入 4.35%, 标签 高收入/高负担消费

建议使用：
- 高收入/低负担消费：可推高客单、品质升级、套装组合，用服务和品质理由让他们花更多，而不是单纯便宜。
- 高收入/高摩擦：先解决选择障碍，再给小额限时激励，重点提高转化率和客单价。
- 低收入/高负担消费：说明消费投入相对重，更适合分期、满减门槛下调、平价替代品和忠诚权益，不能盲目推高价。
- 低收入/低负担消费：不要强推高价商品，优先做内容互动、分享返利和低门槛转化。

## 6. ROI 对比实验设计

实验目标：在相同优惠券预算下，比较传统 RFM 与优化 RFM 的营销 ROI。

实验设置：
- 预算固定：传统策略投放前 20% 高 RFM 用户，共 200 人；单用户券面预算设为 10。
- 传统组：按 RFM_Score 排名前 20% 发券。
- 优化组：只在核心用户、潜力用户、纠结土豪、高潜观望者、高潜流失客中选择，并按预期增量利润排序，人数与传统组一致。

## 7. 结论

传统 RFM 更擅长找到“已经花过钱的人”，但会漏掉高意向低消费、浏览多但未转化、以及高消费但正在流失的人。优化模型通过 I-score、转化摩擦、活跃连接度和购买力背景标签，把优惠券从单纯奖励历史消费，改成优先投向更可能产生增量的人群。

落地优先级：先挽回高潜流失客，再转化纠结土豪和高潜观望者，最后用低成本权益经营隐形活跃者与核心用户。
